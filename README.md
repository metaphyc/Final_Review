- 主观题（已校对）：`主观题_已校对.md`

## “已校对”含义与命名
- “已校对”：仅对结构、排版、标点空格、列表层级、中英对照格式做统一；不做无依据扩写与内容增补。
- 客观题：`Objective_reviewed/客观题_第XX章_已校对.md`
- 主观题：`主观题_已校对.md`

## 排版规范（摘要）
完整规则以 `格式提示词.md` 为准；以下为最常踩坑的统一项：
- 中文冒号：统一写作 `： `（冒号后必须空格）
- 中英对照：中文在前，英文用半角括号置后
- 组织方式：先分段；遇到明显并列关系再用列表（并列很短可行内列举）
- 可读性：段落之间空一行，避免段落与列表挤在一起

## 使用建议
- 复习客观题：从 `Objective_reviewed/` 按章节打开即可。
- 复习主观题：直接阅读 `主观题_已校对.md`，按题号定位。
