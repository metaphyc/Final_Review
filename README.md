# Final Review for Cognitive Neuroscience

## 项目简介
本项目旨在整理和复习认知神经科学相关内容，包含主观题和客观题两部分。

- **主观题**: 记录在 `主观题.md` 文件中。
- **客观题**: 分章节整理，原始版本和美化版本分别存储在 `Objective/` 文件夹中。
- **已校对客观题**: 校对后的客观题存储在 `Objective_reviewed/` 文件夹中。

## 使用说明
1. **查看主观题**: 打开 `主观题.md` 文件。
2. **查看客观题**: 
   - 原始版本：位于 `Objective/` 文件夹中。
   - 美化版本：文件名包含“美化版”。
3. **查看已校对内容**: 打开 `Objective_reviewed/` 文件夹中的文件。
