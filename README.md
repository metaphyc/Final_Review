# 认知神经科学 Final Review

用于整理与复习认知神经科学的题库内容（主观题 + 客观题），并持续将客观题统一格式后“已校对”归档。

## 内容入口
- 主观题：`主观题_已校对.md`
- 客观题（已校对）：`Objective_reviewed/`
- 排版规范（提示词）：`格式提示词.md`
- 参考题库：`题库2025.pdf`

## 目录结构（当前）
```
.
├─ README.md
├─ 主观题_已校对.md
├─ 格式提示词.md
├─ 题库2025.pdf
└─ Objective_reviewed/
   ├─ 客观题_第02章_已校对.md
   ├─ 客观题_第03章_已校对.md
   ├─ ...
   └─ 客观题_第13章_已校对.md
```

## 命名与流程
- 仅当某章客观题完成格式统一与内容核对后，文件名使用 `客观题_第XX章_已校对.md` 并放入 `Objective_reviewed/`
- 未校对/中间态文件不与已校对混放，避免误用
- 全部客观题排版与结构统一遵循 `格式提示词.md`

## 贡献指南
- 尽量只做“排版一致性 + 明显错误修正”，避免无依据扩写
- 提交前检查：标题/题号/来源行、冒号空格、列表层级、中文在前英文在后
