# 客观题解答格式提示词（已校对版）

目标：在不改变内容的前提下，输出与“已校对”文件一致的排版与风格，确保全章统一、可读性强。

0. 基准
- 以同目录下所有“已校对”文件为唯一风格基准
- 仅做排版与结构优化，不补充或删改事实内容

1. 标题与结构
- 顶部标题：# 客观题 · 第X章
- 每题使用二级标题：## 1. 题干
- 题干下紧跟“来源： 第X页…”，并与正文空一行

2. 并列与列表（优先级规则）
- 明确并列关系（如“早期和后期”“背侧与腹侧”）必须列表呈现
- 段落内出现并列内容必须列表化；但仅 2-3 个短词语时改用行内列举
- 只有并列或多项信息时才使用列表；单句描述保持段落
- 多级信息用缩进列表：主类 → 子项 → 例子
- 示例以“例”或“例：”引导，并拆成列表

3. 多小问与小节组织（优先级规则）
- 先分段回答，每一段之间空一行
- 段内出现并列内容才改用列表；不并列则保持段落
- 题干包含多个问题（含“、/及/和”并列小问）且各答案不并列时：使用分段回答，不强制列表
- 同一题内出现“定义/驱动因素/作用”等小节：用分段文字作为小节标题，段与段之间空一行
- 同一题内出现“核心原则/神经机制/实验条件/新发现”等小节：小节标题分段，标题下用列表；若小节下只有一项则用段落
- 避免同一题内“段落+列表”混用导致段落间无空行

4. 标点与空格
- 中文冒号后统一空格：如“术语： 描述”
- 中英文混排空格、标点保持一致；移除多余空格

5. 中英对照与大小写
- 中文在前，英文括号在后
- 英文名大小写统一为“首字母大写 + 其余小写”

6. 语言与颗粒度
- 表述简洁、要点化
- 长句中若含并列成分，拆成列表

7. 一致性检查清单
- 标题、题号、来源行格式是否一致
- 冒号后是否都有空格
- 并列信息是否都用列表或行内列举
- 小节标题之间是否有空行
